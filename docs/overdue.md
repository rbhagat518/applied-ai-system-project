# Managing Overdue Tasks

## Immediate Actions
- **Prioritize Overdue Tasks**: Address overdue tasks as soon as possible.
- **Health Impact Assessment**: Evaluate if delay affects pet's health.
- **Catch-Up Schedule**: Create a plan to complete missed tasks safely.

## Feeding Delays
- If feeding is delayed by less than 2 hours, feed normally.
- For longer delays, consult vet for advice on portion sizes.
- Never double-feed to compensate.

## Medication Delays
- Contact vet immediately for missed doses.
- Some medications have strict timing requirements.
- Never give double doses without veterinary approval.

## Prevention
- Set reminders for important tasks.
- Build flexibility into daily routines.
- Regular schedule reviews help prevent overdue tasks.
